## POWERSHELL COMMAND TO RUN

Copy and paste this command in your PowerShell terminal:

```powershell
# Install SQL Server module (if not already installed)
Install-Module -Name SqlServer -Force -AllowClobber

# Run the database setup script
Invoke-Sqlcmd -ServerInstance "." -Database "master" -InputFile "DATABASE_SETUP.sql"
```

---

## ALTERNATIVE: Using sqlcmd (if you have it installed)

```powershell
sqlcmd -S . -E -i "DATABASE_SETUP.sql"
```

## TROUBLESHOOTING

If you get permission errors:
- Run PowerShell as Administrator
- Make sure SQL Server is running
- Check your SQL Server instance name (default is ".")

If you get connection errors:
- Verify SQL Server is installed and running
- Check if you have SQL Server Management Studio
- Use SSMS to run the script manually instead
